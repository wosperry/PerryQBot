接口列表

1、根据传递参数实现二维码生成

基本信息

Path： http://ip:port/query  (将ip和port使用实际运行服务的IP和端口)

Method： GET

接口描述：

返回结果JSON格式参照1:
{
    "code": 0,
    "msg": "success",
    "result": {
            "base64_image": "data:image/png;base64,iVBORw....",
    }
}

返回结果JSON格式参照2:
{
    "code": 201,
    "msg": "no results",
    "result": null
}

请求参数

Query

参数名称 是否必须 示例 备注
text 是 hello%20world! 生成二维码的内容，需要urlencode
size 否 300 尺寸大小（像素），例如：300
level 否 m 纠错等级，el可用值：h、q、m、l，例如：m
bgcolor 否 ffffff 背景色代码，例如：ffffff
fgcolor 否 000000 前景色代码，例如：000000

返回数据

名称 类型 是否必须 默认值 备注 其他信息
code integer 必须 返回码, 0:表示请求成功 其他表示请求错误
msg string 必须 返回描述
result object 非必须 返回结果, 可能为null
├─ base64_image string 非必须 base64encode编码的二维码图片内容
